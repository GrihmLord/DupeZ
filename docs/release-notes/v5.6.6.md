# DupeZ v5.6.6 — Auto-Update Fix

The auto-updater has been fail-closed since v5.6.2 because the pinned-pubkey list was shipped empty ("intentionally empty — causes the updater to refuse every update") and no release carried the signed `.manifest.json` / `.manifest.sig` sidecar files. v5.6.6 provisions the release keypair, embeds the pinned pubkey, and folds signing into the build pipeline so manifest + signature are emitted on every build.

## Important: one-time manual upgrade required

If you're on **v5.6.5 or older**, your client has no pinned pubkey and will refuse all auto-updates regardless of what we ship. Download `DupeZ_v5.6.6_Setup.exe` manually from this release page. From v5.6.6 onward, auto-update works end-to-end.

This is intentional — trust cannot be bootstrapped from nothing, and we won't retro-add a permissive path to old clients.

## Added

- **Ed25519 release pubkey pinned (`app/core/update_verify.py`).** `TRUSTED_PUBKEYS_PEM` now carries a single PEM-encoded Ed25519 public key. The matching private key is held offline on the maintainer's signing host. Key rotation follows the dual-pubkey transition documented in the module.
- **Signing folded into build pipeline (`packaging/build_variants.bat`).** Build emits `DupeZ_Setup.exe.manifest.json` + `DupeZ_Setup.exe.manifest.sig` automatically when `$env:DUPEZ_SIGN_PRIVKEY` points at the privkey PEM. Skipped gracefully if unset with a clear warning. Both sidecar files must be uploaded as release assets — the updater fetches them by name from the same release.

## Changed

- **Root cleanup.** Moved `RELEASE_NOTES_v5.6.{3,4,5}.md` from the repo root into `docs/release-notes/` (which already held older release notes). Naming standardized to `vX.Y.Z.md`.

## Downloads

- **`DupeZ_v5.6.6_Setup.exe`** — Inno Setup installer (recommended).
- **`DupeZ_Setup.exe`** — stable versionless alias.
- **`DupeZ_Setup.exe.manifest.json`** — signed update manifest. **REQUIRED for auto-update.**
- **`DupeZ_Setup.exe.manifest.sig`** — Ed25519 signature envelope. **REQUIRED for auto-update.**
- **`DupeZ-GPU.exe`** — portable, asInvoker, split-mode, GPU map.
- **`DupeZ-Compat.exe`** — portable, requireAdministrator, in-process engine.

## Test plan
