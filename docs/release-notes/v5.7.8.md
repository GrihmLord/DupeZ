# DupeZ v5.7.8

Packaging hotfix for the frozen Windows builds.

## Fixed

- Bundles `PyQt6.sip` explicitly in PyInstaller builds so `DupeZ-GPU.exe` and `DupeZ-Compat.exe` no longer fail at startup with `ModuleNotFoundError: No module named 'PyQt6.sip'`.
- Makes early launcher output safe in windowed/frozen builds where `sys.stdout` and `sys.stderr` can be `None`.
- Refreshes version metadata for runtime, manifests, installer, and file-version resources.

## Verification

- Source launcher smoke test with `sys.stdout = None` and `sys.stderr = None`.
- Frozen `DupeZ-Compat.exe --verify-self` smoke test.
- Rebuilt v5.7.8 portable artifacts and installer.
